Welcome to My Learning Journal!

originally created as a Scrimba Solo Project, I modified the project and made it my own. I wanted somewhere I could keep track of how I tackled and solved common problems I ran into while working on projects. I also wanted something I could come back too and tweek overtime as my skills grow. While this page is mostly written using HTML and CSS, I do plan on adding more features using JavaScript in the very near future. If you found you're way here from my LinkedIn profile, come back sometime and see what's changed!
